## Key Projects

### CMUX — Multi-Agent Orchestration

A production-grade system for coordinating multiple AI agents working in parallel. Features supervisor hierarchy, worker management, isolated execution environments, automatic rollback on failures, and real-time monitoring. Built to handle complex tasks that require multiple specialized agents collaborating.

### ChainFactory — LLM Pipeline Framework

A framework for building, chaining, routing, and managing LLM calls at scale. Designed for production reliability with built-in error handling, retry logic, and observability. Makes it straightforward to compose complex LLM workflows from simple, testable components.

### FlowPilot — Workflow Automation

An intelligent workflow automation tool that streamlines business processes with smart task routing, conditional logic, and integration with external services.
